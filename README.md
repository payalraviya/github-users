# github-users

## copy .env.example to .env. (skip if already done)

```sh
cp .env.example .env
```

- replace token with your github token

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
